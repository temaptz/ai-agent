# 🚀 НАЧНИТЕ ЗДЕСЬ!

## 🎯 Что это?

Универсальный ИИ-агент для всестороннего анализа вопросов, созданный сеньор-программистом с 20-летним стажем.

## ⚡ Быстрый запуск

### 1. Настройка (5 минут)
```bash
# Установите зависимости
pip install -r requirements.txt

# Настройте Ollama
python setup_ollama.py
```

### 2. Запуск (1 минута)
```bash
# Интерактивный режим
python main.py

# Или конкретный вопрос
python main.py "Как работает машинное обучение?"
```

### 3. Тестирование (2 минуты)
```bash
# Проверьте работу
python test_agent.py

# Посмотрите демонстрацию
python demo.py
```

## 🎭 Что умеет агент?

### 🔍 Глубокий анализ
- Разбирает сложные вопросы на аспекты
- Анализирует мотивацию и контекст
- Выявляет ключевые темы для изучения

### 🌐 Интеллектуальный поиск
- Ищет информацию в интернете через DuckDuckGo
- Загружает и анализирует веб-страницы
- Извлекает релевантную информацию

### 🔄 Циклическая обработка
- Систематически исследует каждый аспект
- Итеративно углубляется в тему
- Адаптивно планирует исследование

### 📊 Синтез результатов
- Объединяет все найденные данные
- Формирует структурированный отчет
- Оценивает уровень уверенности

## 🔧 Настройка

### Изменение модели
```python
# В config.py
ollama_model = "gemma2:9b"  # Или другая модель
```

### Настройка поиска
```python
# В config.py
duckduckgo_max_results = 10  # Количество результатов
request_timeout = 30         # Таймаут в секундах
```

### Ограничение итераций
```python
# В config.py
max_iterations = 5           # Максимум итераций
max_aspects_per_question = 10 # Максимум аспектов
```

## 📊 Результаты

Агент возвращает структурированные результаты:

```json
{
    "original_question": "Ваш вопрос",
    "analysis_results": {
        "motivation": "Анализ мотивации",
        "aspects": ["аспект1", "аспект2"],
        "research_questions": ["вопрос1", "вопрос2"]
    },
    "final_insights": {
        "summary": "Краткий ответ",
        "key_conclusions": ["вывод1", "вывод2"],
        "insights": ["инсайт1", "инсайт2"],
        "recommendations": ["рекомендация1"],
        "confidence": "высокий/средний/низкий"
    }
}
```

## 🚨 Решение проблем

### Ollama не запускается
```bash
ollama serve
```

### Модель не найдена
```bash
ollama pull gemma2:9b
```

### Ошибки поиска
- Проверьте интернет-соединение
- Увеличьте `request_timeout` в конфигурации
