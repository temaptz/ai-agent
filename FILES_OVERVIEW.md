# 📁 Обзор созданных файлов

## 🎯 Основные файлы

### `main.py`
**Главный файл для запуска агента**
- Интерактивный режим работы
- Обработка вопросов из командной строки
- Красивый вывод результатов
- Обработка ошибок

### `config.py`
**Конфигурация приложения**
- Настройки Ollama
- Параметры поиска
- Ограничения агента
- Управление через переменные окружения

### `requirements.txt`
**Зависимости Python**
- LangChain и LangGraph
- Ollama интеграция
- DuckDuckGo поиск
- BeautifulSoup для парсинга

## 🤖 Агент

### `agent/universal_agent.py`
**Основной класс агента**
- UniversalAgent - главный класс
- AgentState - состояние для LangGraph
- Промпты для анализа и синтеза
- Граф состояний с циклами

### `agent/__init__.py`
**Пакет агента**
- Экспорт основных классов
- Инициализация модуля

**Инструменты поиска**

- Форматирование результатов

### `tools/__init__.py`
**Пакет инструментов**
- Экспорт инструментов
- Инициализация модуля
