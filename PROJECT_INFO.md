# 📋 Информация о проекте

## 🎯 Цель проекта

Создание универсального ИИ-агента для всестороннего анализа вопросов с использованием современных технологий Python, LangGraph, LangChain и Ollama с моделью Gemma3.

## 🏗 Архитектура

### Основные компоненты

1. **UniversalAgent** - главный класс агента
   - Использует LangGraph для управления состояниями
   - Интегрирован с LangChain для работы с LLM
   - Поддерживает циклическую обработку аспектов

2. **Инструменты поиска**
   - `WebSearchTool` - поиск в интернете через DuckDuckGo
   - `URLContentTool` - загрузка и анализ веб-страниц

3. **Система промптов**
   - Анализ и разбиение вопросов
   - Исследование конкретных аспектов
   - Синтез результатов

4. **Вспомогательные утилиты**
   - Обработка текста
   - Валидация вопросов
   - Форматирование результатов

## 🔧 Технологический стек

### Основные технологии
- **Python 3.8+** - основной язык программирования
- **LangChain** - фреймворк для работы с языковыми моделями
- **LangGraph** - создание агентов с состояниями и циклами
- **Ollama** - локальный запуск языковых моделей
- **Gemma2:9b** - языковая модель от Google

### Дополнительные библиотеки
- **duckduckgo-search** - поиск в интернете
- **requests** - HTTP-запросы
- **beautifulsoup4** - парсинг HTML
- **pydantic** - валидация данных
- **python-dotenv** - управление переменными окружения

## 🚀 Возможности

### Анализ вопросов
- Глубокий анализ мотивации и контекста
- Разбиение на отдельные аспекты
- Выявление ключевых тем для исследования

### Поиск информации
- Поиск в интернете через DuckDuckGo
- Загрузка и анализ веб-страниц
- Извлечение релевантной информации

### Циклическая обработка
- Систематическое исследование каждого аспекта
- Итеративное углубление в тему
- Адаптивное планирование исследования

### Синтез результатов
- Объединение всех найденных данных
- Формирование структурированного отчета
- Оценка уровня уверенности в выводах

## 📊 Метрики качества

### Производительность
- Время обработки: 30 секунд - 10 минут
- Количество аспектов: 3-15
- Источников на аспект: 5-20

### Качество анализа
- Уровень уверенности: высокий/средний/низкий
- Полнота охвата темы
- Релевантность источников

## 🔄 Процесс работы

### 1. Анализ вопроса
```
Вопрос → Анализ мотивации → Разбиение на аспекты → Планирование исследования
```

### 2. Исследование аспектов
```
Аспект → Поиск информации → Анализ данных → Формирование выводов
```

### 3. Синтез результатов
```
Все аспекты → Объединение данных → Структурирование → Финальный отчет
```
